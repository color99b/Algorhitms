# 🔵 Priority Queue

🍊 Heap의 자료구조에 FIFO를 적용한 선형 자료구조이다.  
-> Heap을 상속하여 아래 3가지의 method만 구현해주면 된다.

- enqueue : Heap에 data 삽입
- dequeue : Heap의 data 삭제
- isEmpty : Heap이 비어있나 확인

```javascript
enqueue = (data) => this.push(data);
dequeue = () => this.poll();
isEmpty = () => this.heap.length <= 0;
```
